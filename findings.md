# Findings — Vaporwave Theme

## Color Palette
- Pink: #FF6B9D (neon pink)
- Cyan: #00D4FF (bright cyan)  
- Purple: #B44DFF (vivid purple)
- Blue: #4D7CFF (electric blue)
- Dark bg: #1A0A2E (deep purple-black)
- Sunset orange: #FF6B35

## Fonts Available
- PixelifySans-Medium.ttf — retro pixel font, good for vaporwave title
- Tektur-Medium.ttf — geometric tech font
- JetBrainsMono-Bold.ttf — monospace terminal feel

## Vaporwave Design References
- Greek statues (often bust + sunglasses) — core meme
- Palm trees + grid floor + sunset — classic "Outrun" aesthetic
- Floppy disks, dolphins, geometric shapes — 90s retro
- Flat 2D style, high saturation, clear outlines
